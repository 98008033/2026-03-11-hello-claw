## 🎮 项目简介

Hello Claw 是一个交互式点击计数应用，具有华丽的连击特效系统。

**核心功能：**
- 点击按钮增加计数
- 连击系统（combo）
- 5 秒倒计时自动重置
- 每 10 连击背景色轮换
- 50/100 连击特殊特效

## 🚀 快速开始

### 本地开发

**启动后端：**
```bash
cd backend
cargo run
```

**启动前端：**
```bash
cd frontend
python3 -m http.server 8080
```

访问 http://localhost:8080

### Render 部署

**自动部署：**
1. Push 到 GitHub main 分支
2. Render 自动触发构建
3. 前端自动注入 API 地址

**手动部署：**
1. 打开 https://dashboard.render.com
2. 找到 hello-claw-frontend
3. 点击 Manual Deploy
