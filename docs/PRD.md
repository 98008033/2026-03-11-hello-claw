## 1. 产品概述

### 1.1 产品定位

一个带有动态计数效果的前端页面，后端使用 Rust + SQLite 持久化存储计数数据。

### 1.2 目标用户

- 需要简单计数工具的用户
- 想体验炫酷点击效果的用户

### 1.3 核心价值

- 🎨 视觉爽感 - 点击按钮时有丰富的动画特效
- 💾 数据持久化 - 计数不会丢失
- ⚡ 高性能 - Rust 后端保证响应速度

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 计数显示 | 展示当前计数值 | P0 |
| 点击计数 | 点击按钮增加计数 | P0 |
| 数据持久化 | 计数保存到数据库 | P0 |
| 重置计数 | 清空计数器 | P1 |

### 2.2 交互需求

| 需求 | 描述 |
|------|------|
| 按钮点击效果 | 波纹动画 + 粒子飞散 |
| 数字动画 | 计数更新时缩放弹跳 |
| 连击系统 | 快速点击显示 Combo |
| 键盘支持 | 空格键可触发点击 |

---

## 3. 技术需求

### 3.1 后端

- 语言：Rust
- 框架：Actix-web
- 数据库：SQLite
- 端口：10000
- 监听：0.0.0.0

### 3.2 前端

- 技术栈：HTML5 + CSS3 + Vanilla JS
- 动画：CSS Animations + JavaScript
- 响应式：支持移动端

### 3.3 API 接口

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/count` | GET | 获取当前计数 |
| `/api/increment` | POST | 增加计数 |
| `/api/reset` | POST | 重置计数 |
